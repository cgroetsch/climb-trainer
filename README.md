# Climb Trainer Pro 🧗

Training-App für Tim — Weg zur NK2 (U17)

## Features
- 📊 Dashboard mit Live-Statistiken
- 🏔️ Aktivitätskalender (GitHub-Style, letzte 12 Wochen)
- 💪 Workout-Tracker (mit Swipe-to-Delete)
- 🍎 Ernährungstracker mit Kalorien/Makros
- 📈 Stats & Milestones
- 🗑️ Swipe-to-Delete für alle Einträge

## Tech Stack
- Vanilla HTML/CSS/JS (kein Framework)
- Supabase (PostgreSQL Backend)
- Netlify Hosting

## Live
https://climb-trainer-tim.netlify.app

## Setup
1. Supabase-Projekt anlegen
2. SQL-Schema aus `schema.sql` ausführen
3. `SUPABASE_URL` und `SUPABASE_ANON_KEY` in `index.html` anpassen
4. Auf Netlify deployen
